# Customer Churn Prediction (Machine Learning)

This project predicts whether a telecom customer will churn (leave) using a baseline Machine Learning model.

## What I did
- Loaded and cleaned the dataset
- Converted categorical features using one-hot encoding
- Trained a Logistic Regression model
- Evaluated results using accuracy and classification report

## Tech Stack
- Python, Pandas, NumPy
- Scikit-learn
- Matplotlib
